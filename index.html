Dogenode (January 2014 - April 2014)

Despite the name, Dogenode is a powerful social networking web application with a functional REST API. Dogenode talks to other social networks through REST interfaces, which allowed Dogenode users to befriend and follow users across different social networks. Dogenode users could also subscribe to their GitHub activity. This web application was built using SQLite, Django, Python, HTML5, CSS (Bootstrap, custom), and JS (jQuery, custom). Screenshots of the main features are below:

  • Make a New Post
  • My Stream
  • GitHub Events in My Stream
  • Friends Management
  • Posts Management
  • Search for Authors

402 Story Map (January 2014 - April 2014)

The first Agile story map tool for GitHub projects, currently being tested by CS undergraduates at the University of Alberta. 402 Story Map was built using Node.js (Express), HTML5, CSS (Bootstrap, custom), and JS (jQuery, Handlebars.js, Routie, nProgress.js, custom). The web application is mostly client-side, while the server is responsible for OAuth authentication and wrapping GitHub API calls. The web application supports HTTPS if an SSL certificate is supplied to the server.

American Sign Language - Phonological Awareness Test (ASL-PAT) (January 2012 - April 2012)

In cooperation with the Minerva Deaf Research Lab (MDRL) and fellow undergraduates, a first-of-its-kind web application for measuring American Sign Language (ASL) proficiency was developed using MySQL, PHP, HTML5, CSS (Bootstrap), and JS (custom). The project has since continued on without me, but here are some screenshots of how the web application looked after 3 months of development:
